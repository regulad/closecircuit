<p>Close as in close-by, not close as in shut down. The name is a play on CCTV.</p><p>This project lets you create a PAN surveillance system using WiFi IP Webcams. The monitor functions as an AP using the Wi-Fi Direct APIs, and can have multiple Wi-Fi cameras connected to it.</p><p>The terminal monitor runs the CC app (which you are currently looking at), and the cameras are other Android devices running the Android IP Webcam app.</p>
